Eco Developer Guide

Amazon Alexa is a voice interactive platform that we used to develop our own skill. The skill includes: providing user eye-opening sustainability facts, providing users tips on sustainable living at home, work and school, providing user information on how to recycle certain materials, and provide user the closest recycling centers.

Features

  1. Interact with user to answer their question related to sustainability.
  2. Add most frequently asked question on website, so we can update our database by adding more information. And make it available on website for user.
  3. Provide "eco" challenges, which are tasks that we can assign to users to teach them how to live more sustainably.

How to Install

  1. Download Alexa application.
  2. Search for Alexa skills on the Amazon website.
  3. Enable Skill by saying enable “skill name”.
  4. To Enable Alexa skill:
    1. Select the menu on skills, or go to the Alexa skills store on the Amazon website: https://www.amazon.com/skills.
    2. Open the skill detail when you find a skill.
    3. Select the Enable skill option, or ask Alexa to open the skill.

Components and Technologies used

  1. Hardware: - Alexa capable device
  2. Service: - AWS Lambda
  3. Database: - Dynamodb
  4. Languages: - Node.js, HTML, CSS

How to contribute?

All the files and code will be posted on our GitHub page.
